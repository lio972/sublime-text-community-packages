The Merge Plugin

Merge plugin 

This command will diff/merge the last two selected files.

You need to have WinMerge installed: http://winmerge.org/

Default shortcut is Ctrl+Alt+D.

To use: click on view A, click on view B, press Ctrl+Alt+D.
WinMerge will be launched with files A and B.
 

To Install This Plugin

  1. download the Merge Plugin to your local hard-drive.
  2. double-click the package file
  3. restart Sublime Text

For Developers

  • This source code for this plugin is available at google code.
  • Google provides instructions for checking out the code
