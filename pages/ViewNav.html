The ViewNav Plugin

ViewNav Plugin for Sublime Text Editor

ViewNav - Goodness for Navigating View

ViewNav is a handy plugin that adds a little extra awesomeness for moving and navigating around groups/views using some simple key bindings while you have multiple views/panes open in Subime Text.

action

| value | binding |description | | focus | alt+arrow | makes a different group (pane) active | | move | alt+shift+arrow | move the current view (buffer) to a different group (pane) |

destination

| value | destination | | (default) | next group | | right | next group | | down | next group | | next | next group | | left | previous group | | up | previous group | | previous | previous group |

To Install This Plugin

  1. download the ViewNav Plugin to your local hard-drive.
  2. double-click the package file
  3. restart Sublime Text

For Developers
