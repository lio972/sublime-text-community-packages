Exports the document/selection[s] as a html+css representation of the current view / theme

[http://www.sublimetextwiki.com/resources/html_export.py.html](http://www.sublimetextwiki.com/resources/html_export.py.html)

# Usage         

ctrl+alt+shift+e,ctrl+alt+shift+e: export current document / selection[s] 

ctrl+alt+shift+e,ctrl+alt+shift+l: ditto + with line numbers


# Multiple Selections
            
Will export multiple selections separated with ... ellipsis 

eg. 

    baz = Foo() * Bar()                    

    ...                                    

    print baz                              

    ...
