Readme
======

This plugin exports two commands:

* `ctrl+alt+f4`, `quickClose`,       Close multiple files using `ctrl+enter` with the QuickPanel
* `ctrl+shift+d`, `closeAndDelete`,  Close and delete a file with one key-stroke.  (Binding commented out by default)

Dependencies
============
*    AAALoadFirstExtensions
