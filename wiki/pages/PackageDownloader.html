The PackageDownloader Plugin

Downloads any package at http://www.sublimetextwiki.com.

Install it (see below for instructions) and there will now be a menu item to download packages at Tools | Packages | Package Downloader | Browse Packages. Choose a package to download. You should not need to restart Sublime Text -- the plugin should start working straight away.

To Install This Plugin

  1. download the PackageDownloader Plugin to your local hard-drive.
  2. double-click the package file
  3. restart Sublime Text

For Developers

  • This source code for this plugin is available at google code.
  • Google provide Instructions for checking out the code
