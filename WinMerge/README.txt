This plugin will open WinMerge to merge two files together.

Configuration
-------------

Edit the `winmerge.py` file and set this constant ( if needed )

WINMERGE = '"C:\Program Files\WinMerge\WinMergeU.exe"'

There are also other configuration options. Check them out.

Usage
-----

Run the command on the first file you wish to merge and then the next file you `activate` (click | ctrl+tab) will be opened along with the first in WinMerge.
